## Project setup
```
// 基于vue-cli，请先全局安装vue-cli：
yarn global add @vue/cli

// 安装apicloud-cli:
yarn global add apicloud-cli

// 安装本项目的依赖
yarn
```

### Compiles and hot-reloads for development
```
yarn run serve

```
yarn run build
